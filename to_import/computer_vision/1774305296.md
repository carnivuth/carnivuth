[CNN](1774305280.md) to recongnize handwritten digits.

![](/Pasted-image-20241001101124.png)

Principal features are:

- $5\times 5$ kernels
- decrease of spatial dimension and an increase of layers dimension alongside the layers
- no padding, [average pooling](1774305280.md#POOLING-LAYERS)
- 2 [fully connected layers](1774305282.md#FULLY-CONNECTED-LAYERS) where each unit compute the distance between the input and the parameter vector
